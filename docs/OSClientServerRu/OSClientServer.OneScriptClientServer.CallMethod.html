КлиентСерверДляОдноСкрипта.ВызватьМетод (OneScriptClientServer.CallMethod) Метод

Синтаксис

ВызватьМетод(<Сценарий>, <ИмяМетода>, <МассивПараметров>)

Параметры

Сценарий (обязательный)
Тип: Произвольный.
Сценарий, как объект с типом значения ScriptEngine.Machine.IRuntimeContextInstance.
ИмяМетода (обязательный)
Тип: Строка.
Имя вызываемого метода.
МассивПараметров (обязательный)
Тип: Массив.
Массив передаваемых в метод параметров.

Возвращаемое значение

Описание

Вызывает методы сценариев по их именам.

На самом деле это вызов одноименного метода экземпляра класса Рефлектор из стандартной библиотеки классов и функций односкрипта. Используется для упрощения кода.
