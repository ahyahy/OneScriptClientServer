Действие.Сценарий (Action.Script) Свойство

Использование

Чтение и запись.

Значение

Тип: Произвольный.

Возвращает или задает сценарий как объект с типом значения ScriptEngine.Machine.IRuntimeContextInstance.

Примечание

Пример

Действие = КС.Действие(ЭтотОбъект, "TCPСервер1_ПриОтключенииКлиента");
TCPСервер1.ПриОтключенииКлиента = Действие;

Сообщить("Действие.Сценарий = " + ТипЗнч(Действие.Сценарий));
