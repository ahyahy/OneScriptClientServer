Обзор ПриложениеСервис (ServiceApplication)

Конструктор

ПриложениеСервис (ServiceApplication) Конструктор Создаёт новый экземпляр класса ПриложениеСервис (ServiceApplication).

Свойства

Клиенты (Clients) Массив объектов с типом значения ПриложениеСервисКлиент (ServiceApplicationClient), подключенных к серверу приложений.
ПриОтключенииКлиента (ClientDisconnected) Возвращает или задает код для выполнения, когда клиент отключается от сервера.
ПриПодключенииКлиента (ClientConnected) Возвращает или задает код для выполнения, когда клиент подключается к серверу.
Результат (Resalt) Возвращает или задает результат выполнения метода приложения (скрипта) в контексте сервера.

Методы

ВыполнитьНаКлиенте (DoAtClient) Выполняет на клиенте процедуру или функцию с именем, переданном в параметре ИмяМетода.
Начать (Start) Запускает ПриложениеСервис (сервер).
Остановить (Stop) Останавливает ПриложениеСервис (сервер).
