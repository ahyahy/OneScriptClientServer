Конструктор

TCPКлиент (TcpClient) Конструктор Создаёт новый экземпляр класса TCPКлиент (TcpClient).

Свойства

ПриОтключении (Disconnected) Возвращает или задает код для выполнения при отключении от сервера.
ПриОтправкеСообщения (MessageSent) Возвращает или задает код для выполнения при отправке сообщения без какой-либо ошибки.
ПриПодключении (Connected) Возвращает или задает код для выполнения при подключении к серверу.
ПриПолученииСообщения (MessageReceived) Возвращает или задает код для выполнения при получении нового сообщения.
СостояниеСоединения (CommunicationState) Возвращает текущее состояние соединения.

Методы

Отключить (Disconnect) Выполняет отключение клиента.
ОтправитьСообщение (SendMessage) Отправляет сообщение на сервер.
Подключить (Connect) Выполняет подключение клиента.
