СерверКлиент.ИдентификаторКлиента (ServerClient.ClientId) Свойство

Использование

Только чтение.

Значение

Тип: Число.

Уникальный идентификатор для этого клиента на сервере.

СерверКлиент = КС.СерверКлиентАрг().Клиент;
Сообщить("Идентификатор = " + СерверКлиент.ИдентификаторКлиента);
