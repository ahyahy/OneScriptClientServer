Обзор СерверКлиент (ServerClient)

Свойства

ИдентификаторКлиента (ClientId) Уникальный идентификатор для этого клиента на сервере.
ПриОтключении (Disconnected) Возвращает или задает код для выполнения, при отключении клиента от сервера.
СостояниеСоединения (CommunicationState) Возвращает текущее состояние соединения.

Методы

Отключить (Disconnect) Выполняет отключение от клиента и закрывает базовый канал связи.
ОтправитьСообщение (SendMessage) Отправляет сообщение клиенту.
