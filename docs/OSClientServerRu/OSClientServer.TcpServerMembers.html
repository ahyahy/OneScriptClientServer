Обзор TCPСервер (TcpServer)

Конструктор

TCPСервер (TcpServer) Конструктор Создаёт новый экземпляр класса TCPСервер (TcpServer).

Свойства

Клиенты (Clients) Набор клиентов, подключенных к серверу.
ПриОтключенииКлиента (ClientDisconnected) Возвращает или задает код для выполнения, когда клиент отключается от сервера.
ПриОтправкеСообщения (MessageSent) Возвращает или задает код для выполнения при отправке сообщения без какой-либо ошибки.
ПриПодключенииКлиента (ClientConnected) Возвращает или задает код для выполнения, когда новый клиент подключается к серверу.
ПриПолученииСообщения (MessageReceived) Возвращает или задает код для выполнения при получении нового сообщения.

Методы

Начать (Start) Запускает сервер.
Остановить (Stop) Останавливает сервер.
