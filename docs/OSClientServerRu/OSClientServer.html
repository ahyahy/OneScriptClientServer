Классы

Класс Описание
TCPКлиент (TcpClient) Представляет клиента для подключения к серверу.
TCPКонечнаяТочка (TcpEndPoint) Представляет конечную точку TCP.
TCPСервер (TcpServer) Представляет сервер, который используется для приема клиентских подключений и управления ими.
АргументыСобытия (EventArgs) Представляет базовый класс для классов, содержащих данные событий, и предоставляет значение для событий, не содержащих данных.
ВыполнитьНаСервереАрг (DoAtServerArgs) Хранит имя метода, который нужно вызвать на сервере.
КлиентСерверДляОдноСкрипта (OneScriptClientServer) Объект КлиентСерверДляОдноСкрипта (OneScriptClientServer) разработан для создания клиент-серверных приложений.
ПриложениеКлиент (ServiceClient) Представляет клиента, который может вызывать методы приложения, находящегося на сервере.
ПриложениеКлиентАрг (ServiceClientEventArgs) Хранит информацию о клиенте сервиса, которая будет использоваться событием.
ПриложениеСервис (ServiceApplication) Представляет сервер, обеспечивающий функциональность приложения на компьютере сервера.
СерверКлиент (ServerClient) Представляет клиента с точки зрения сервера.
СерверКлиентАрг (ServerClientEventArgs) Хранит информацию о клиенте, которая будет использоваться событием.
СообщениеАрг (MessageEventArgs) Предоставляет данные для событий отправки/получения сообщений.
СообщениеБайты (ByteMessage) Представляет сообщение используемое для отправки/получения необработанного массива байтов в качестве данных сообщения.
СообщениеБулево (BoolMessage) Представляет сообщение используемое для отправки/получения значения булево в качестве данных сообщения.
СообщениеДата (DateTimeMessage) Представляет сообщение используемое для отправки/получения даты в качестве данных сообщения.
СообщениеТекст (TextMessage) Представляет сообщение используемое для отправки/получения текста в качестве данных сообщения.
СообщениеЧисло (NumberMessage) Представляет сообщение используемое для отправки/получения числа в качестве данных сообщения.
