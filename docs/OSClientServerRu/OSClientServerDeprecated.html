Устаревшие свойства, методы

В целях совместимости устаревшие свойства и методы будут работать как и прежде. Рекомендуется использовать новый синтаксис.

В других особых случаях будут даны разъяснения.


Устаревшее Рекомендуемое
Установка обработчика события
Начиная с версии OneScript 1.8.3 обработчик события библиотеки OneScriptClientServer нельзя устанавливать используя класс Действие (Action) из библиотеки OneScript.
Так не сработает: 

ПриложениеКлиент1.ПриПодключении = Новый Действие(ЭтотОбъект, "ПриложениеКлиент1_ПриПодключении");
Начиная с версии OneScriptClientServer 2.0.1.0 обработчик события библиотеки OneScriptClientServer нужно устанавливать используя только класс Действие (Action) из библиотеки OneScriptClientServer.
Используйте: 

ПриложениеКлиент1.ПриПодключении = КС.Действие(ЭтотОбъект, "ПриложениеКлиент1_ПриПодключении");
В случае запуска библиотеки OneScriptClientServer на версии OneScript ниже версии 1.8.3 этой рекомендации можно не придерживаться.
