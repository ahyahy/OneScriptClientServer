TCPКлиент (TcpClient) События

События, поддерживаемые библиотекой для класса TCPКлиент (TcpClient), и ссылка на свойство задающее обработчик события, перечислены ниже. Полный список членов класса TCPКлиент (TcpClient) смотрите в разделе TCPКлиент (TcpClient) Члены.

События

ПриОтключении (Disconnected) Происходит при отключении от сервера.
ПриОтправкеСообщения (MessageSent) Происходит при отправке сообщения без какой-либо ошибки.
ПриПодключении (Connected) Происходит при подключении к серверу.
ПриПолученииСообщения (MessageReceived) Происходит при получении нового сообщения.
