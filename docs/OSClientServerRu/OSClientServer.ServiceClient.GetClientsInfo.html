ПриложениеКлиент.ПолучитьИнформациюКлиентов (ServiceClient.GetClientsInfo) Метод

Синтаксис

ПолучитьИнформациюКлиентов()

Параметры

Возвращаемое значение

Тип: Массив.

Массив экземпляров класса ИнформацияКлиента (ClientInfo).

Описание

Возвращает массив экземпляров класса ИнформацияКлиента (ClientInfo), содержащих данные о клиенте приложения.

Примечание

Пример

Массив = ПриложениеКлиент1.ПолучитьИнформациюКлиентов();
