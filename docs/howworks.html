Обратите внимание, что можно создать два разных сервера.

  • TCPСервер (TcpServer) для взаимодействия с клиентом TCPКлиент (TcpClient), нативным клиентом TCPСоединение / TCPConnection из библиотеки односкрипта и браузер-клиентом.
  • ПриложениеСервис (ServiceApplication) для взаимодействия с клиентом ПриложениеСервисКлиент (ServiceApplicationClient).

    • У этих серверов разное назначение и функциональность. Они запрограммированы для работы только со своими клиентами.

    На основе TCPСервер (TcpServer) можно организовать взаимодействие компьютеров в сети.

    На основе ПриложениеСервис (ServiceApplication) можно создать сетевое приложение с графическим интерфейсом для каждого клиента. Пример смотрите в справке в разделе Дополнения-Заготовка для чата (Использовалась библиотека Графический интерфейс для сценарного языка OneScript и дизайнер форм в придачу ).
